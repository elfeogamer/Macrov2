📋 Ticket Maestro — Plantilla de Orden
¿Qué es el Ticket Maestro?
Es la plantilla de referencia con todos los productos en el orden que tú defines. Cuando generes los tickets por ruta, cada uno respetará exactamente este orden — si la ruta no lleva un producto, simplemente se omite, pero la posición de los demás no cambia.

⠿ Arrastra productos o categorías para reordenarlos.
➕ Agrega productos extra que quieras incluir siempre.
💾 Guarda — se mantiene aunque cierres el navegador.
💡 Tip: Sube primero el archivo OLA en la pestaña Tickets para que el maestro se construya automáticamente con todos los productos. Luego regresa aquí a ordenarlos.
